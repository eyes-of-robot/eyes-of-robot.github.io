基础知识总结

Author: hzn Date: October 8, 2024  21:38:06 Category: Other

java -jar 制作与执行

  • 新建ClassA.java文件
  • 新建ClassB.java
  • 生成class文件
  • 创建META-INF/MENIFEST.MF文件
  • 生成jar包
  • 执行

注意：jar后面不用加主类了，因为MENIFEST.MF中已经有Main-Class了

java -cp使用

  • 新建ClassA.java文件
  • 新建ClassB.java
  • 生成class文件
  • 执行java -cp命令

注意：其中ClassB为含有Mai方法的类名，-cp后面需要加上所有依赖的class获取jar；格式为.:/home/file/ 路径后面不用带*号

注意：java -jar和java -cp不能同时使用
